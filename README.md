# 💊 مُذكِّر الأدوية | Medication Reminder

> تطبيق ويب بسيط يساعد المرضى وكبار السن على عدم نسيان مواعيد أدويتهم.

## 🌐 جرب التطبيق مباشرة

**[🔗 افتح التطبيق](https://amani-alzyoud.github.io/medication-reminder/)**

## 🎯 المشكلة التي يحلها

كبار السن والمرضى المزمنين (سكري، ضغط، قلب) كثيراً ما ينسون مواعيد أدويتهم، مما يؤثر سلباً على صحتهم. هذا التطبيق يحل المشكلة بطريقة بسيطة وفعّالة.

## ✨ المميزات

- ✅ إضافة أدوية مع الوقت والجرعة والملاحظات
- 🔔 إشعارات على الجهاز (Desktop & Mobile Notifications)
- 🗣️ تنبيه صوتي باللغة العربية يقول: "حان موعد دوائك، قم بتناوله الآن"
- 🔊 صوت جرس تنبيه
- 💾 حفظ البيانات تلقائياً (LocalStorage)
- 📱 يعمل على الموبايل والكمبيوتر
- 🎨 واجهة عربية جميلة RTL

## 🚀 طريقة التشغيل محلياً

1. حمّل الملفات: `index.html`, `style.css`, `script.js`
2. ضعها في مجلد واحد
3. افتح `index.html` في متصفح Chrome
4. اضغط **🔔 تفعيل الإشعارات** والموافقة
5. أضف أدويتك واترك التبويب مفتوحاً

## 🛠️ التقنيات

- HTML5 / CSS3 / JavaScript
- Web Notifications API
- Web Speech API
- LocalStorage
- Web Audio API
